Задание 3

  1. HTML

    HTML — стандартизированный язык разметки документов для просмотра веб-страниц в браузере.
    Веб-браузеры получают HTML документ от сервера по протоколам HTTP/HTTPS или открывают
    с локального диска, далее интерпретируют код в интерфейс, который будет отображаться на экране монитора.

  2. CSS

    CSS — формальный язык описания внешнего вида документа, написанного с использованием языка разметки.
    Также может применяться к любым XML-документам, например, к SVG или XUL.

  3. JavaScript

    JavaScript — мультипарадигменный язык программирования. Поддерживает объектно-ориентированный,
    императивный и функциональный стили. Является реализацией спецификации ECMAScript.
    JavaScript обычно используется как встраиваемый язык для программного доступа к объектам приложений.
